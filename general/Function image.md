>[!tldr] Function image
>Given a [[Function|function]] $f: A \rightarrow B$ the *image* of the function is $F(A) = \{f(a) \vert a \in A\} \subset B$. Note this is not necessarily equal to the [[Function codomain|codomain]] $B$.
