>[!tldr] Internet Protocol Stack (IPS) 4 layers
>The IPS stack is a model of how communication should work over the internet. It is structured into 4 different layer:
>1. Application layer
>2. Transport Layer
>3. Internet Layer
>4. Network access Layer
>
