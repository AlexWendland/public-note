## Overview

As a Software Engineer at [Man Group](https://www.man.com/), I work within the Codex team, which operates the primary alternative data onboarding platform for the company. Our platform manages hundreds of ETL pipelines running daily that support critical trading strategies.

## Technical Environment

**Platform & Tools**
- Custom ETL platform for data processing at scale
- Custom data catalogue for metadata management
- Proprietary mapping solutions for data transformation
- Event-driven architecture for data quality

## Key Contributions

**Data Quality Framework**
- Built out an event-driven data quality framework
- Implemented monitoring and validation across hundreds of pipelines

**Data Infrastructure Modernization**
- Led migration from ArcticDB to Apache Iceberg for improved data lakehouse capabilities
- Enhanced query performance and data versioning

**Cross-Platform Integration**
- Built integrations from Python to C# using Apache Flight Server
- Enabled seamless data access across different technology stacks

## Skills Developed

This role has significantly deepened my understanding of data systems at scale, including the challenges of managing high-volume, high-frequency data pipelines in a production environment.
