>[!definition] Routing table
>Routers keep a table of where [packets](packets.md) [IP addresses](internet_protocol_(ipv4).md) addresses belong to which connection for redirecting messages. This table maps the tuple of an [Internet Protocol (IPv4)](internet_protocol_(ipv4).md) and a [network mask](network_mask.md) to either an interface or a [Internet Protocol (IPv4)](internet_protocol_(ipv4).md). You can use [Route summarization](route_summarization.md) to keep a routing table short.
