>[!tldr] Function image
>Given a [function](function.md) $f: A \rightarrow B$ the *image* of the function is $F(A) = \{f(a) \vert a \in A\} \subset B$. Note this is not necessarily equal to the [codomain](function_codomain.md) $B$.
