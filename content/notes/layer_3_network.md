>[!definition] Layer 3 Network
> This layer works with [nodes](node_(ipv6).md) identified by an [IP addresses](internet_protocol_(ipv4).md). This applies a header to a [Segment](segment.md) with the source and destination [Internet Protocol (IPv4)](internet_protocol_(ipv4).md). Devices at level 3 are:
>- [hosts](host_(networks).md), or
>- [Routers](router.md).
