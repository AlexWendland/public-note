>[!tldr] Logical or
>This is a [boolean function](boolean_function.md) regularly expressed as $\lor: \{T, F\}^2 \rightarrow \{T, F\}$ which is only false if both its inputs are.
>
>This can be generalised to more inputs where all inputs must be False for it to be false.
