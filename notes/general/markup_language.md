# Markup Language

A [Markup Language](markup_language.md) is text-encoding system that tells the computer how to display some text. There are 3 main categories
- **Presentational markup** - ironically the least well presented of all of them. Essentially binary code that is interpreted for programs such as Word.
- **Procedural markup** - This uses tags within the language to define how it should be rendered. Such as [Markdown](markdown.md) and [Tex](tex.md).
- **Descriptive markup** - (Sometimes called semantic) This defines how to label sections of the text. Such as [Latex](latex.md) and [HTML](html.md).

# Difference from a [Programming Language](programming_language.md)

A [Markup Language](markup_language.md) is a [Declarative Language](declarative_language.md) instead of a [Programming Language](programming_language.md) which is an [Imperative Language](imperative_programming.md). This means is specifies an end state rather than a sequence of instructions to get there
