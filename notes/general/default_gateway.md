>[!tldr] Default Gateway
> The default gateway for a [network](network.md) is a [Gateway](gateway.md) of a [router](router.md) on the network which this [host](host_(networks).md) will route [Packets](packets.md) that have an [Internet Protocol (IPv4)](internet_protocol_(ipv4).md) that is not in this [Network](network.md) (as given by the [Network mask](network_mask.md)).
