>[!tldr] Step function methods
>Suppose we have a function $f: \mathbb{R} \rightarrow \mathbb{R}$ a step function method uses [step function](step_function.md) to model this. Here we choose intervals of the [domain](function_domain.md) to define different functions on each region.
