>[!tldr] Binary step
>The *binary step* is an [activation function](activation_function.md) that determines what the value is in reference to a parameter $\theta$.
>$$a_{\theta}(x) = \begin{cases} 1 & \mbox{if } x \geq \theta\\ 0 & \mbox{if } x < \theta . \end{cases}$$
