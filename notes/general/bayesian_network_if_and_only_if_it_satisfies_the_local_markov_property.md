# Statement

> [!important] Lemma
> Let $G = (V,E)$ be a [directed acyclic graph](directed_acyclic_graph_(dag).md) and $X = \{X_v\}_{v \in V}$ a set of [random variables](random_variable.md). $(G,X)$ is a [Bayesian network](bayesian_network.md) if and only if it satisfies the [Local Markov property](local_markov_property.md).
